Installation

You need to install package files/dependencies for this project if you want to customize it. To do this, you must have node and npm installed on your computer. Installation guide of the node can be found here. As npm comes bundled with a node, a separate installation of npm is not needed. If you have installed them, just go to the root folder and run the following command in your command prompt or terminal.

npm i

This will install the dev dependencies in the local node_modules folder in your root directory. The other thing that Gulp requires, which, is really what does all the work, is the gulpfile.js. In this file, you set up all of the tasks that you will run. To run this project in development mode enters the following command in your command prompt or terminal.

npm run gulp

To edit or customize .sass code in your project enters the following command in your command prompt or terminal.

npm run gulp sass:watch

To edit or customize .pug code in your project enters the following command in your command prompt or terminal.

npm run gulp pug:watch
